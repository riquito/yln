# @riquito/yln

`yln` — a small CLI to manage `yarn link` relationships across local projects.

## Install

```sh
npm install -g @riquito/yln
```

This pulls in a tiny per-platform package containing the prebuilt binary for your OS and CPU (Linux x64/arm64, macOS x64/arm64, Windows x64). All other platform packages are skipped by npm.

## Usage

Run `yln` for the full help text. See the [project README](https://github.com/riquito/yln#readme) for details.
